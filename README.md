# Romaji Cli

Get romaji pronunciation for Japanese input (hiragana/katakana).

This is a self-use small tool for learning Japanese pronunciation.

## Installation

```shell
cargo build
```

## Usage

```shell
romaji -i おはよう
```

Output be like:

```txt
Get romaji pronunciation for おはよう ...

o ha yo u
お は よ う
```
